# Requirements

- NodeJS 18+
- Vite 5+
- Vue 3+
- Pinia 2+
- TailwindCSS 3+
- TypeScript 5+
- Vue Router 4+
- Android Studio
- Cordova (npm install -g cordova)
- Java JDK 8
- Gradle 7.6

# Build

```bash
pnpm build
```

# Preview App

```bash
pnpm preview
```

# Build Android

```bash
pnpm build-android
```

# Inject App on Android device

```bash
pnpm inject-android
```
